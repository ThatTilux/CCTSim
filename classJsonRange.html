Create a vector of random uvw configs for a pathconnect2 node.

Parameters
pathconnect2The pathconnect2 node.
num_configsThe number of configs to create. Must be at least 1.
Returns
The vector of configs.

Create a set of configs from a pathconnect2 node with respect to the lower and upper bounds lb and ub. The first config in the returned vector is the default config used when use_previous is disabled, see pathconnect2_default_config(). It is advisable to set symmetric, enable_w and the number of control points before calling this function as the format of the configs depend on these values.
