enum HarmonicScalingFunctionTarget

Enum for the type of the targeted scaling function value for custom CCT harmonics.

Enum to specify the type of the targeted scaling function value for custom CCT harmonics. For harmonics with an amplitude of 'const', the target is the single constant value. For harmonics with an amplitude of 'linear', the target can be the offset or slope value of the scaling function.
