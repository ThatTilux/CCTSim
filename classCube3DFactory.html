Constructs a Cube3DFactory object using two Plane Calculations from RAT-GUI.

Parameters
length1The length1 value of both planes (mm).
length2The length2 value of both planes (mm).
offset_xThe offset x value of both planes (mm).
offset_y_plane_bottomThe offset y value of the bottom plane (mm).
offset_y_plane_topThe offset y value of the top plane (mm).
offset_zThe offset z value of both planes (mm).
invert_cubeIf true, the cube is considered inverted, meaning the cube spans across the entire coordinate space, excluding the area between the two planes.

To visually set a cube, use two Plane Calculations (top and bottom) in RAT-GUI with normal=y. Use the same length1, length2, offset x and offset z but different offset y settings in the two planes to span the 3D cube between the two planes.
