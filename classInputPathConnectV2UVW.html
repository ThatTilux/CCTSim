Detailed Description

Class for defining the uvw1 and uvw2 parameters of a pathconnect2 node as the input parameter.

Construct a InputPathConnectV2UVW object.

Parameters
JSON_nameThe 'name' field of the pathconnect2 node (rat::mdl::pathconnect2).
x_configsThe x vectors of the uvw configurations to be applied.
pathconnect2The pathconnect2 node to be configured.
column_name_suffixThe suffix to be added to the column name in the output file. Default column name is 'pathconnect2_uvw'.

Initialize an InputPathConnectV2UVW object to define the uvw1 and uvw2 configurations of a pathconnect2 node. Format for items of x_configs, see see rat::mdl::PathConnect2:get_x0(). Values of x_configs are to be provided in [m]. Optimization variables (e.g., is_symmetric, enable_w) will be taken from the pathconnect2 node, thus it must be fully configured beforehand.
